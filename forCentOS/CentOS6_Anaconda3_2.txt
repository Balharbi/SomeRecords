1. Install CentOS 6.

2. Install Anaconda3：
　　bash Anaconda3-4.4.0-Linux-x86_64.sh　　　
　　logout
　　which python
　　which pip
　　python -V
   Creat symbol link for pip3, conda3, anaconda3:
         cd /home/pengyong/anaconda3/bin 
         ln -s pip pip3 　## 对空格数目敏感。
         ln -s conda conda3
         ln -s anaconda anaconda3
   

3. Install Anaconda2:
   bash Anaconda2-4.4.0-Linux-x86_64.sh   -b   -p  $HOME/anaconda3/envs/Anaconda2
   logout
   which python
   which python3
   which pip
   which pip3
   which conda
   which conda3
   which anaconda
   which anaconda3
   

４. Change the versions of Anaconda and python:
   conda info --envs    # conda info -e
   source activate  Anaconda2
   source deactivate  Anaconda2

5. Install some softwares by using anaconda3:
   anaconda3 search -t conda gcc
   anaconda  show  TimVideos/gcc-or1k-elf-newlib
   conda install --channel https://conda.anaconda.org/TimVideos gcc-or1k-elf-newlib
                   
   anaconda  show  brown-data-science/vim 
   conda install --channel https://conda.anaconda.org/brown-data-science vim
   anaconda  show  ActivisionGameScience/zsh
   conda install --channel https://conda.anaconda.org/ActivisionGameScience zsh
